¿Qué es un System Prompt?

Ejercicio: System Prompt

Pasos

  1. Abre tu workflow
  2. Doble click en AI Agent
  3. Escribe tu System Prompt
  4. Prueba enviando "Hola"

Ejemplo de Prompt

// System Prompt ejemplo
Eres un asistente amable que ayuda
con preguntas sobre cocina.

Reglas:
- Responde siempre en español
- Sé conciso (máx 3 oraciones)
- Si no sabes, di "No estoy seguro"
JSON y Datos en n8n

Tips de System Prompt

Tip

Sé específico con el rol, el tono y las restricciones. Un buen System Prompt define qué hace, cómo responde y qué NO debe hacer tu agente.

Memoria

¿Qué es la Memoria?

Démosle memoria a nuestro agente

Pasos

  1. Buscar Window Buffer Memory
  2. Conectar al puerto Memory
  3. windowSize = 10
  4. sessionId = {{ $json.message.chat.id }}
  5. Probar: "Me llamo [nombre]" → "¿Cómo me llamo?"

Sobre la Memoria

Tip: Costos

Más memoria = más tokens enviados al LLM en cada mensaje. Un windowSize de 10 es un buen balance entre contexto y costo. Ajusta según tu caso de uso.
