🤖 Elige tu tipo de Chatbot

Cada estudiante puede elegir entre 4 tipos de chatbots. Todos comparten la misma arquitectura base pero se especializan en diferentes casos de negocio.

🍽️ Bot de Reservas

Reserva mesas, citas y eventos en tiempo real. Perfecto para el sector restaurantero y de eventos.

Herramientas

  • Calculator: Precio total de reserva
  • HTTP Request: API Open-Meteo (clima)
Ver Guía Paso a Paso →

🛒️ Bot de E-commerce

Soporte al cliente: estado de pedidos, stock de productos y cálculo de descuentos.

Herramientas

  • Calculator: Descuentos para clientes
  • HTTP Request: API de stock (JSON)
Ver Guía Paso a Paso →

📚️ Bot Educativo

Tutorías personalizadas: explica conceptos, proporciona ejercicios y tracking de progreso.

Herramientas

  • Calculator: Horas totales del curso
  • HTTP Request: API de cursos (temario)
Ver Guía Paso a Paso →

🎲️ Bot de Entretenimiento

Diversión y curiosidades: datos curiosos, chistes, gatitos y trivia.

Herramientas

  • Calculator: Streak de días consecutivos
  • HTTP Request: Cat Facts API (GRATIS)
Ver Guía Paso a Paso →

🏗️ Arquitectura Base (Común a todos)

[Telegram Trigger] → [AI Agent (Tools Agent)]
                                         ↓
               [System Prompt personalizado]
                                         ↓
               [Simple Memory] (window size 10)
                                         ↓
               [Calculator Tool] + [HTTP Request Tool]
                                         ↓
                                         [Respuesta por Telegram]

✅ Nodos n8n Nativos Utilizados

  • ✅ Telegram Trigger - Punto de entrada, dinamismo en clase
  • ✅ Chat/AI Agent - Cerebro con system prompt personalizado
  • ✅ Simple Memory - Window buffer, mantiene contexto
  • ✅ Calculator - Herramienta nativa, matemáticas exactas
  • ✅ HTTP Request - Herramienta nativa, APIs externas
  • ✅ OpenAI Chat Model - gpt-4o-mini, barato y rápido

🎯 Ventajas de esta arquitectura

  1. Misma arquitectura → Los estudiantes aprenden una vez
  2. Casos de negocio reales → Cada uno aplica a diferentes sectores
  3. Solo nodos nativos → Sin complejidades, HTTP requests mínimos
  4. Telegram优先 → Dinamismo en clase, feedback instantáneo
  5. Videos impresionantes → Estilo startup >$1B USD
