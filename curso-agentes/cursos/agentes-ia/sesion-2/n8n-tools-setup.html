Parte B

HTTP Request Tool

Conecta con el mundo

Busca HTTP Request Tool

IMPORTANTE: Busca 'HTTP Request' y selecciona el que dice 'Tool'

HTTP Request Tool

Atencion

Hay 2 nodos HTTP Request. El normal es una accion. El Tool es para que el agente decida cuando usarlo.

Conecta al puerto Tool

Igual que Calculator, conectalo a otro puerto Tool del AI Agent

Configuracion (opcional)

Puedes dejarlo sin configurar para que el agente use cualquier URL, o fijar una URL especifica:

URL: https://api.chucknorris.io/jokes/random

Tip

Sin URL fija, el agente puede llamar cualquier API que le indiques en el chat
