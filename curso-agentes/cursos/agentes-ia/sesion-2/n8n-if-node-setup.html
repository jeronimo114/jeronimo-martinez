Decisiones automaticas en tu workflow

Resumen

4 pasos

1
Buscar nodo IF
2
Colocar entre Telegram y AI Agent
3

4

1

Busca el nodo IF

En el panel de nodos escribe 'IF'

IF
2

Coloca entre nodos

El IF va ENTRE Telegram Trigger y AI Agent. Desconecta la linea directa y coloca el IF en medio

Flujo

Telegram → IF → AI Agent

3

Configura la condicion

Doble click en el IF. Configura:

Value 1: {{ $json.message.text }}
Operation: Contains
Value 2: urgente
Si el mensaje contiene 'urgente', va por True
4

Conecta la ruta True

La salida 'true' (verde) va al AI Agent. Arrastra una linea desde la salida verde al AI Agent

True

True = la condicion se cumplio

5

Conecta la ruta False

La salida 'false' (roja) puede ir a un nodo de respuesta simple o a otro flujo

Por ejemplo, un nodo 'Respond to Webhook' que diga 'Mensaje recibido, no es urgente'
Referencia

Operaciones disponibles

1
Contains - contiene texto
2
Equals - es exactamente igual
3
Greater Than - mayor que (numeros)
4
Is Empty - esta vacio
5
Regex - patron avanzado

Prueba el IF

Envia 'Esto es urgente' → debe ir al AI Agent

Envia 'Hola' → debe ir por la ruta False
