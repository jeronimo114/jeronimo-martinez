1. Introducción

Objetivo: Crear un tutor personalizado de cursos.

Arquitectura:

Telegram Trigger → AI Agent → Calculator + HTTP Request → Telegram Response
↓
Simple Memory

2. Setup Inicial

Prerrequisitos:

  • Cuenta en n8n.cloud
  • Token de Telegram
  • Datos de cursos (simulados en JSON)

Crear Workflow en n8n:

  1. Nuevo workflow vacío
  2. Guardar como: Bot-Educativo-[tu-nombre]

3. Paso 1: Telegram Trigger

Acción: Añadir nodo Telegram Trigger

Configuración:

Bot Token: [pegar tu token de @BotFather]

Test:

  1. Click "Execute Node"
  2. Envía un mensaje: "Hola" a tu bot en Telegram
  3. Debería ver el trigger en n8n

4. Paso 2: AI Agent + System Prompt

Acción: Añadir nodo Chat (AI Agent)

Configuración:

Resource: gpt-4o-mini

System Prompt:
Eres TutorBot, tutor personalizado de cursos de programación y tecnología.

Idioma: español

Tono: paciente, motivador, claro y pedagógico

Funciones:
- Explicar conceptos técnicos de forma accesible
- Dar ejemplos prácticos y del día a día
- Proponer ejercicios para reforzar aprendizaje
- Adaptar explicaciones según nivel del estudiante
- Ofecer motivación y ánimo constante
- Recomendar recursos adicionales cuando sea apropiado

Reglas:
- Si el estudiante no entiende, reformula usando analogías simples
- Usa ejemplos concretos antes de teoría abstracta
- Mantén explicaciones breves (máximo 100-150 palabras)
- Nunca juzgues ni te frustes: reconoce cuando algo está fuera de tu conocimiento
- Ofrece ánimo y motívación: "¡Excelente pregunta!", "Lo estás haciendo muy bien"
- Si el estudiante está atascado, ofrece: ¿Qué tal si lo explicamos de otra manera?
- Siempre termina con un follow-up: ¿Te queda claro? ¿Tienes alguna otra duda?

Memory Configuration:

Memory Type: Simple Memory
Session ID: {{ $json.message.chat.id }}
Window Size: 15 (mayor para contexto educativo)

Test:

  1. Ejecuta el workflow
  2. Conversa con el bot
  3. Verifica que recuerda el progreso del estudiante

5. Paso 3: Simple Memory

El nodo Simple Memory ya está configurado en el paso anterior. Este nodo mantiene los últimos 15 mensajes de la conversación, permitiendo un mayor contexto educativo.

6. Paso 4: Calculator Tool (Progreso)

Acción: Añadir nodo Calculator

Configuración:

Operation: Multiplicación
Input 1: Número de módulos
Input 2: Horas por módulo

Tool Name: calculate_total_hours
Tool Description: Calcula total de horas del curso

Test:

  1. Pregunta: "¿Cuántas horas tiene el curso?"
  2. Debería responder: "12 horas" (4 módulos × 3h = 12h)

7. Paso 5: HTTP Request (Cursos)

Acción: Añadir nodo HTTP Request con JSON simulado

Configuración:

Crear nodo "Set" con este JSON como valor inicial:
{
  "cursos": {
    "python-intro": {
      "name": "Introducción a Python",
      "modules": ["Variables", "Loops", "Funciones"],
      "duration": 12,
      "hours_per_module": 3,
      "level": "principiante",
      "description": "Aprende desde cero con proyectos prácticos"
    },
    "javascript-basics": {
      "name": "JavaScript Básico",
      "modules": ["Variables", "Funciones", "DOM", "Eventos"],
      "duration": 20,
      "hours_per_module": 4,
      "level": "intermedio",
      "description": "Crea interfaces web interactivas"
    },
    "react-fundamentals": {
      "name": "React Fundamentals",
      "modules": ["Componentes", "Props", "State", "Hooks"],
      "duration": 30,
      "hours_per_module": 6,
      "level": "avanzado",
      "description": "Construye aplicaciones modernas con React"
    }
  }
}

Tool Name: get_course_info
Tool Description: Obtiene información de curso

Test:

  1. Pregunta: "¿Qué cursos tienen?"
  2. Debería listar los 3 cursos disponibles

8. Testing Completo

Escenarios de Prueba:

✅ Escenario 1: Recomendación de curso

Usuario: ¿Qué curso me recomiendas para empezar programación?
Bot: Para empezar, te recomiendo Introducción a Python. Es ideal para principiantes.

✅ Escenario 2: Detalles del curso

Usuario: ¿De qué trata?
Bot: Contiene 4 módulos: Variables, Loops, Funciones, y Proyectos Prácticos. Dura 12 horas.

✅ Escenario 3: Dificultad del curso

Usuario: ¿Qué tan difícil es?
Bot: Nivel principiante. Perfecto si nunca has programado. ¿Te gustaría ver el temario detallado?

✅ Escenario 4: Segunda pregunta con memoria

Usuario: ¿Es difícil?
Bot: [Debe recordar que estábamos hablando del mismo curso]
