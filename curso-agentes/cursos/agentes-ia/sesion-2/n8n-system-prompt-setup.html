Abre tu Workflow

El que tiene el nodo AI Agent conectado a Telegram

Hint: Si no lo tienes, crealo primero

2

Doble click en AI Agent

Se abre el panel de configuracion

Tip

El nodo AI Agent es el cerebro de tu chatbot

3

Busca System Message

En la parte superior del panel, veras un campo grande de texto

El campo se llama System Message

4

Escribe tu Prompt

Ejemplo de System Prompt

Eres ESIC-Bot, un asistente de marketing educativo. Reglas: - Responde siempre en espanol - Se conciso (maximo 2 parrafos) - Si no sabes algo, dilo honestamente - Usa un tono profesional pero amigable
Template

Template recomendado

Estructura base para tu System Prompt

Eres [NOMBRE], un [ROL]. Tu objetivo es [OBJETIVO]. Reglas: - Idioma: [IDIOMA] - Tono: [TONO] - Limite: [LIMITE DE RESPUESTA] - Si no sabes: [QUE HACER]

Mejores practicas

Itera y mejora
