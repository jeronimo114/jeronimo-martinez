1

Busca Window Buffer Memory

En el panel de nodos, escribe 'Window Buffer Memory'

Window Buffer Memory
2

Arrastra al canvas

Colocalo debajo del nodo AI Agent

Tip

Debe estar cerca del AI Agent para conectar facil

3

Conecta al puerto Memory

El AI Agent tiene un puerto inferior llamado 'Memory'. Arrastra una linea desde Window Buffer Memory hasta ese puerto

Tip

La linea debe ser del puerto de salida al puerto Memory

4

Configura windowSize

Doble click en Window Buffer Memory. Cambia 'Window Size' a 10

Window Size: 10

El bot recordara los ultimos 10 mensajes de cada conversacion

5

Configura sessionId

En el campo Session ID, escribe:

{{ $json.message.chat.id }}

Esto separa la memoria por cada usuario de Telegram

Contexto

Por que importa

1
Sin memoria → el bot olvida todo entre mensajes
2
Con memoria → mantiene el contexto de la conversacion
3
sessionId → cada usuario tiene su propia memoria
