¿Que es un LLM?

🧠

Solo cerebro

  • Modelo de lenguaje (GPT, Claude, Gemini)
  • Solo genera texto
  • No tiene memoria entre sesiones
  • No puede ejecutar acciones
LLM vs Agente

¿Que es un Agente?

🤖

Cerebro + Cuerpo

  • LLM como cerebro
  • Herramientas para actuar
  • Memoria persistente
  • Autonomia para decidir

Respuestas vs Acciones

LLM vs Agente

¿Cuando usar cada uno?

Usa LLM cuando...

  • Necesitas generar texto
  • Quieres analizar informacion
  • Buscas respuestas puntuales
  • No requieres acciones externas

Usa Agente cuando...

  • Necesitas ejecutar tareas
  • Quieres automatizar procesos
  • Requieres memoria a largo plazo
  • Buscas autonomia real

Conectar Sheets

Misma cuenta de Google

Credentials → Add Credential → Google Sheets OAuth2

Proyecto 1 | Paso 4

Gmail Trigger

Configuracion

  • Agregar nodo "Gmail Trigger"
  • Event: "Message Received"
  • Credential: Seleccionar
  • (Opcional) Filtrar por label

Estructura del email

{ "from": { "value": [{ "address": "sender@mail.com" }] }, "subject": "Asunto", "date": "2024-01-15" }
Proyecto 1 | Paso 5

Google Sheets Node

Configuracion

  • Agregar nodo "Google Sheets"
  • Operation: Append Row
  • Spreadsheet: "Registro de Emails"
  • Sheet: "Sheet1"

Mapeo de campos

Remitente: {{ $json.from.value[0].address }} Asunto: {{ $json.subject }} Fecha: {{ $json.date }}

Debugging
