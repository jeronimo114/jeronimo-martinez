Solo cerebro

  • No tiene memoria entre sesiones
  • No puede ejecutar acciones

🤖

Cerebro + Cuerpo

  • LLM como cerebro
  • Herramientas para actuar
  • Memoria persistente

¿Cuándo usar cada uno?

Usa LLM cuando...

  • Necesitas generar texto
  • Quieres analizar información
  • Buscas respuestas puntuales
  • No requieres acciones externas

Usa Agente cuando...

  • Necesitas ejecutar tareas
  • Quieres automatizar procesos
  • Requieres memoria a largo plazo
  • Buscas autonomía real

¿Qué es un MCP?

Model Context Protocol

Estándar abierto creado por Anthropic para conectar LLMs con herramientas externas.

  • Un "USB universal" para IAs
  • Cualquier app puede crear un MCP
  • El LLM decide cuándo usarlo
  • Seguro y controlado

Ejemplos de MCPs

📂 Filesystem - Leer/escribir archivos

🔍 Brave Search - Buscar en internet

🐙 GitHub - Gestionar repos

📊 Google Sheets - Manipular datos

🗃️ PostgreSQL - Consultar DBs
