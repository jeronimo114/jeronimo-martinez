# Demo 2: Prompts - Vago vs Preciso

## Información General

| Aspecto | Detalle |
|---------|---------|
| **Duración** | 5 minutos |
| **Módulo** | 1 - Fundamentos LLM |
| **Slide** | S1-11 |
| **Herramienta** | ChatGPT (chat.openai.com) |

## Objetivo del Demo

Demostrar visualmente la diferencia en calidad de respuesta entre un prompt vago y uno preciso, creando un momento "aha" para la audiencia.

---

## Preparación Pre-Demo

### Checklist
- [ ] ChatGPT abierto en pestaña
- [ ] Dos pestañas/conversaciones nuevas listas
- [ ] Zoom al 125% para visibilidad
- [ ] Copiar prompts en notas para pegar rápido

### Prompts a Usar

**Prompt Vago:**
```
Escríbeme algo sobre automatización
```

**Prompt Preciso:**
```
Escribe 3 beneficios de automatizar emails de bienvenida para tiendas ecommerce.

Formato: bullets de 1 línea cada uno
Tono: profesional pero accesible
Audiencia: dueños de tiendas online sin conocimientos técnicos
```

## Script del Demo

### Introducción (30 segundos)

> "Vamos a hacer un experimento en vivo. Le voy a dar a ChatGPT dos instrucciones diferentes sobre el mismo tema, y quiero que vean la diferencia."

### Paso 1: Prompt Vago (1.5 minutos)

1. Abrir primera conversación de ChatGPT
2. Pegar: `Escríbeme algo sobre automatización`
3. Enviar

**Mientras genera:**
> "Fíjense qué tan general es mi instrucción. No le dije cuánto, en qué formato, para quién..."

4. Cuando termine, leer por encima:
> "Miren esto. Un ensayo largo, genérico, que podría servir para cualquier cosa... o para nada en particular."

**Señalar problemas:**
- Demasiado largo
- Sin estructura clara
- Podría ser de Wikipedia
- No es accionable

### Paso 2: Prompt Preciso (2 minutos)

1. Abrir segunda conversación
2. Pegar el prompt preciso
3. Enviar

**Mientras genera:**
> "Ahora le estoy dando: cantidad específica, formato, tono, y audiencia. Todo en 4 líneas."

4. Cuando termine, mostrar con entusiasmo:
> "¡Miren esto! Exactamente 3 puntos, en bullets, claros y accionables."

**Señalar mejoras:**
- Cantidad exacta solicitada
- Formato que pedí
- Lenguaje apropiado para la audiencia
- Útil inmediatamente

### Paso 3: Comparación Visual (1 minuto)

1. Poner ambas respuestas lado a lado (o alternar)
2. Señalar contraste:

> "La diferencia es dramática, ¿verdad? Y las dos vienen del mismo modelo, el mismo ChatGPT, en el mismo momento."
>
> "La única diferencia fue cómo le pedí las cosas."

## Puntos Clave a Enfatizar

### El Modelo No Lee Tu Mente
> "ChatGPT no sabe que quieres bullets hasta que se lo dices"

### La Precisión Ahorra Tiempo
> "Con el prompt vago, tendría que editar y pedir cambios. Con el preciso, está listo para usar."

### El Template
Mostrar en pantalla:
```
Escribe [CANTIDAD] [TIPO] sobre [TEMA]
para [AUDIENCIA], en formato [FORMATO], con tono [TONO].
```

---

## Transición al Ejercicio

> "Ahora les toca a ustedes practicar. Van a tomar un prompt vago y transformarlo en uno preciso usando el template que vimos."
>
> "Tienen 8 minutos. Luego compartimos las mejores versiones."

## Variaciones del Demo

### Si el tiempo es corto (3 min)
- Saltar el prompt vago
- Mostrar solo el preciso
- Explicar qué habría pasado sin precisión

### Si hay preguntas durante
- "Excelente pregunta, la respondemos después del ejercicio"
- Mantener momentum del demo

### Si ChatGPT está lento
- Tener screenshots de backup
- O usar respuestas pre-generadas

## Respuestas Esperadas

### Prompt Vago - Típica Respuesta
```
La automatización es el proceso de utilizar tecnología
para realizar tareas con mínima intervención humana...

[Sigue por 3-4 párrafos sobre historia, tipos,
beneficios generales, futuro de la automatización...]
```

### Prompt Preciso - Típica Respuesta
```
3 beneficios de automatizar emails de bienvenida para ecommerce:

• Aumenta conversiones: Los clientes que reciben email de
  bienvenida compran 3x más en su primera semana.

• Ahorra tiempo: Una vez configurado, funciona 24/7 sin
  intervención manual.

• Mejora experiencia: El cliente se siente valorado desde
  el primer momento.
```

## Troubleshooting

### Si ChatGPT no está disponible
- Usar Claude, Gemini, o cualquier LLM
- El principio es el mismo

### Si la respuesta del prompt vago es buena
- "Incluso cuando sale bien, no podemos predecirlo"
- "La consistencia viene de la precisión"

### Si la respuesta del preciso no es perfecta
- "Aún así, está mucho más cerca de lo que necesito"
- "Puedo iterar desde aquí más fácilmente"

## Notas del Instructor

### Timing Detallado
- Intro: 0:30
- Prompt vago + análisis: 1:30
- Prompt preciso + análisis: 2:00
- Comparación + cierre: 1:00
- **Total: 5 minutos**

### Tips de Presentación
- Mostrar frustración (actuada) con la respuesta vaga
- Mostrar satisfacción con la respuesta precisa
- Hacer que la audiencia sienta la diferencia

### Errores a Evitar
- No leer toda la respuesta vaga (aburre)
- No explicar demasiado el template
- No hacer más de 2 ejemplos
